# 🚗 Car Rental Fullstack Web Application (MERN Stack)

A full-stack car rental platform built using the **MERN stack** where car owners can list their cars for rent and users can book cars easily.

---

## 📌 Project Overview

This project is a **car rental marketplace** that connects car owners with customers who want to rent cars.
Owners can upload their cars, and users can browse available cars and rent them through the platform.

---

## ✨ Features

### 👤 User Features

* User authentication (Register / Login)
* Browse available cars
* View car details
* Rent cars
* Responsive UI

### 🚘 Owner Features

* List cars for rent
* Manage car listings

### ⚙️ General Features

* Full-stack MERN architecture
* RESTful APIs
* Modern UI with Tailwind CSS
* Client-server architecture

---

## 🛠 Tech Stack

### Frontend (Client)

* React.js (Vite)
* Tailwind CSS
* React Router DOM
* Axios
* React Hot Toast

### Backend (Server)

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication (if implemented)

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Abhishek-dev69/CarRental-fullstack.git
cd CarRental-fullstack
```

---

### 2️⃣ Setup Backend (Server)

```bash
cd server
npm install
npm start
```
---

### 3️⃣ Setup Frontend (Client)

```bash
cd client
npm install
npm run dev
```
